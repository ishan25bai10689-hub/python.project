#  Personal Health Check System

A simple Python-based health analysis tool that calculates BMI, daily nutrition needs, and provides personalized health suggestions based on user input.

---

##  Problem Statement

Many students and individuals are unaware of their daily nutritional needs, BMI status, and overall health condition. Most available tools are either complex or not personalized.

This project aims to provide a **simple, interactive, and easy-to-use health checker** that gives meaningful insights using basic inputs like weight, height, age, and lifestyle.

---

##  Features

- Calculates **BMI (Body Mass Index)**
- Suggests **ideal body weight**
- Provides **daily calorie needs**
- Calculates **protein, carbs, and fat requirements**
- Recommends **water intake and sleep duration**
- Suggests **vitamin intake based on gender**
- Generates a **health score**
- Gives **personalized fitness suggestions**:
  - Weight gain
  - Weight loss
  - Maintenance

---

##  Technologies Used

- Python (Core Concepts)
- Conditional Statements
- User Input Handling
- Basic Health Calculations
